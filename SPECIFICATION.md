# Overview

Create a modern website for the fictional **SIT Encoding Standard**.

The project is intentionally a parody but must look like a real technical standard.

The design should resemble official organizations such as:

- IEEE
- ISO
- RFC Editor
- MDN
- W3C

The user should initially think that SIT is a real encoding standard before slowly realizing it is satire.

The website must be fully responsive.

No backend.

Everything runs in the browser.
